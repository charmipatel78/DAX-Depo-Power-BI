# 📊 DAX Depo - Power BI Project

## 📌 Overview

This project demonstrates the use of DAX (Data Analysis Expressions) in Power BI to perform calculations, create measures, and extract meaningful insights from data.

---

## 🎯 Objective

* To understand and apply DAX functions
* To create calculated columns and measures
* To perform data analysis using DAX
* To enhance analytical capabilities in Power BI

---

## 📂 Project Files

* `DAX Depo.pbix` – Power BI project file
* `DAX_Depo_Sample_Datasets.xlsx` – Sample dataset used for analysis

---

## ⚙️ Key Features

* Use of DAX functions
* Creation of calculated columns
* Development of measures
* Data aggregation and analysis
* Logical and conditional expressions

---

## 🧮 Sample DAX Measures

```DAX
Total Sales = SUM(Sales[Amount])

Average Sales = AVERAGE(Sales[Amount])

Sales Category = 
IF(Sales[Amount] > 1000, "High", "Low")
```
